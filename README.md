# NotesHub

## 📚 Project Overview

NotesHub is a web-based Notes Sharing Application developed using PHP and MySQL. It allows users to upload, view, and download study notes in PDF, JPG, and JPEG formats.

---

## Features

- Upload notes
- Enter Topic Name
- Enter Subject Name
- View uploaded notes
- Download notes
- File type validation
- File size limit (5 MB)
- Responsive user interface

---

## Installation

1. Install XAMPP.
2. Copy the project into htdocs.
3. Start Apache and MySQL.
4. Create the database.
5. Import the SQL file.
6. Open:

http://localhost/NotesHub
